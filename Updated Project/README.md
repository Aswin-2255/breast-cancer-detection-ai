# Breast Cancer Detection using Machine Learning

This project predicts whether a tumor is malignant or benign using machine learning and a Flask web application.

## Features
- Machine learning classification model
- Breast cancer prediction
- Flask web interface
- Model evaluation using confusion matrix

## Technologies Used
- Python
- Scikit-Learn
- Pandas
- NumPy
- Flask

## Dataset
Wisconsin Breast Cancer Dataset

## How to Run

1. Install libraries
pip install -r requirements.txt

2. Run the application
python app.py

3. Open the browser and access the web interface
